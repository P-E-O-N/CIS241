Reflections -

Matthew Flack




As I think back on the 10 weeks of this course I had quite a few “Ah ha” moments where something that was giving me difficulty got a bit easier. A lot of them were SUPER basic things but as this is my very first experience with HTML I think that’s ok. The CIW website does a pretty good job of introducing the various concepts but I don’t think it goes into them in very much depth. That is probably a good thing because I don’t retain a lot of detailed information like that without applying it a few times first.



  • My first “Ah ha” moment was when I came across the www.w3schools.com resource. I’ve been using that website as my foremost place to look for additional information since I discovered it.

  • Another such moment was realizing that for myself I REALLY learn best by doing. This is probably truer in the case of web development, but sadly it took me a few weeks before I realized this. I needed to sit down pull up Notepad++ and explore any idea / concept that I was having trouble with.

  • “Ah Ha”… If you right click on a website the “inspect” option has a TON of information in it. I was aware of the ability to view source, but some-how had never just clicked on inspect and I regret that it took me so long.

  • CSS box model (content / padding / border / margin) and the general idea that the basic layout of a website consists of containers within containers within containers. I watched MANY different videos on the application of those ideas. I feel like I understood the basics of using any given <element>, but the general layout was difficult.

  • Another time where I felt like I was suddenly able to grasp a concept a lot more fully, was after reading this detailed description that I came across on how floats work. The following link is an explanation of floats and I still find myself reading over this occasionally because I don’t understand it on an intuitive level yet.

  • CSS Selectors: I wanted practice using the various selectors and came across this little game, which helped clarify them. I think I’m on level 20 something. CSS Diner

  • GIMP can do everything I need to do with graphics for producing my website (so far). I hadn’t played around with GIMP before but during the creation of my site I found I needed to add transparency layers, edit the size of images, or use the transform property to orient things as desired. I had to watch quite a few "how to" video's but GIMP is free and got the job done.

  • Github only likes lowercase letters because it's Linux based. I know this was specifically covered in the homework where we uploaded stuff to Github but it seems like it caused me grief each and every time and was always a case of lowercase vs uppercase.

  • My final “Ah ha” moment was more of a ……. “Ahhhhhhhh” moment when I realized that I was finally ready to turn in my competency demonstration. I realize there is no end to the leaning though, so I better get used to it!
